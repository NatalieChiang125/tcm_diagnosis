# TCM Diagnosis (中醫診斷網站)

這是一個簡單的 **中醫診斷網站**，用於模擬中醫問診與診斷建議。使用者可以輸入症狀，系統會提供相應的中醫辨證結果。  
專案主要是 **HTML / CSS / JavaScript** 前端實作，並部署於 GitHub Pages。

📌 線上 Demo: [https://nataliechiang125.github.io/tcm_diagnosis/](https://nataliechiang125.github.io/tcm_diagnosis/)

---

## 功能 Features

- 使用者輸入症狀與狀態
- 系統提供中醫辨證建議
- 簡單直覺的表單介面
- 可立即查看診斷結果
- 完全前端實作，無後端依賴

---

## 技術棧 Tech Stack

- **前端**: HTML, CSS, JavaScript
- **部署**: GitHub Pages
